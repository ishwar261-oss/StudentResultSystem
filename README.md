# Student Result System (Java)

A simple Java-based console application that generates a complete student marksheet. It calculates total marks, percentage, CGPA, grade, and pass/reappear status for five subjects using clean OOP structure.

## Features
- Accepts student details and subject-wise marks
- Calculates:
  - Total marks (out of 500)
  - Percentage
  - CGPA
  - Subject-wise Grade
  - Subject-wise Result (Pass / Reappear)
- Automatically assigns overall result
- Clean, formatted console output
- Uses object-oriented programming principles

## Technologies Used
- Java
- OOP Concepts
- Scanner Class
- Formatted Printing (printf)

## Project Structure
StudentRes.java          → Class for calculation and display  
useofStudentRes.java     → Main class for input and execution

## How to Run
1. Install Java (JDK 8 or above)
2. Compile both files:
   javac StudentRes.java useofStudentRes.java
3. Run the program:
   java useofStudentRes

## Purpose
This project demonstrates:
- Java OOP implementation
- User input handling
- Conditional logic
- Console formatting

Useful for beginners, students, and portfolio/resume projects.
